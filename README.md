# Blossom
SDDM Waifu

Prerequisites
KDE Plasma Desktop

Dependencies

* "SDDM"


# Install

* copy the blossom folder to the following directory:

```
   /usr/share/sddm/themes/
```

And add "blossom" to you 
/etc/sddm.conf



```
[Theme]
# Current theme name
Current=blossom
```
